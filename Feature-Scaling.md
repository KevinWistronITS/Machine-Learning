1. Standardization
    
    ![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/501f9335-3ea5-4a78-8f58-2a9a485d1c59/Untitled.png)
    
- Your value should be between 3 and -3.
    
    [-3;3]
    

For example,

|  | Salary (USD) | Age (yrs) |
| --- | --- | --- |
| A | $70000 | 45 |
| B | $60000 | 44 |
| C | $52000 | 40 |

After normalization:

|  | Salary (USD) | Age (yrs) |
| --- | --- | --- |
| A | 1 | 1 |
| B | 0.44 | 0.8 |
| C | 0 | 0 |
